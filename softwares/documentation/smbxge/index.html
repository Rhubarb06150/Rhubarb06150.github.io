Super Mario Bros. X Graphic Editor Documentation


How does it works?

SMBX Graphic Editor is open source, it's made with Python 3 and use the Tkinter Framework for GUI,
and PIL and opencv for image processing.

What are features?

SMBX Graphic Editor permits you to manage, see and modify custom graphics for a
SMBX 1 or SMBX 2 level, you can even preview animated graphics.

__________ Help __________

Everytime you open a level folder, the .lvl or .lvlx file is backuped in the "backup" folder,
located in the root folder of the software, which prevent an sudden file deletion or corruption.
